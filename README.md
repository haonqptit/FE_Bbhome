# FE_Bbhome

Frontend cho Hotel Booking Portal BB Homes, xây dựng bằng Next.js App Router, TypeScript và Tailwind CSS.

## Chạy môi trường development

Yêu cầu Node.js `>=20.9.0` và pnpm `10.34.5`.

```bash
pnpm install
pnpm dev
```

Truy cập `http://localhost:3000/vi` hoặc `http://localhost:3000/ja`.

## Kiểm tra dự án

```bash
pnpm lint
pnpm typecheck
pnpm build
```
